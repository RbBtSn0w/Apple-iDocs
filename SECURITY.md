# Security Policy

## Supported Versions

We provide security updates for:

- `main` branch
- latest published release

Older releases may not receive security fixes.

## Reporting a Vulnerability

Please report security issues privately.

- Email: **security@idocs.dev**
- Preferred language: Chinese or English
- Include: affected version/commit, impact, reproduction steps, and any proof of concept

## Response Targets

- Initial acknowledgment: within 3 business days
- Triage decision: within 7 business days
- Fix timeline: depends on severity and release window

Please do not disclose vulnerabilities publicly before a fix is available.
